# Yoshi# CLI
This allows you to code with Yoshi-related keywords, and allows a yoshi-run command to be used on files with the .yoshi extension!
This will then run via C# (as it is translated to C# keywords.)
Note that this is mostly made for fun, and is not really expected to be used for formal or professional type projects.
